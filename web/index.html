实验室简介

互联网创新实验室（原复杂系统与电子商务创新实验室），以“互联网+”为核心，以 web 系统开发、电子商务案例深度分析、网站设计， UI 设计、网站易用性研究，网络营销、 MVC 模式系统开发、 UI 设计 SNS 等方面为研究方向。目前实验室分成程序、前端、产品三个方向，并通过项目开发为导向进行实践提升。

10年来，实验室作为培养互联网精英的摇篮，采用项目驱动制因材施教模式，运行实现常态化，前期毕业成员广泛分布于阿里巴巴、腾讯、新浪、百度、360、网易、金山等大型互联网公司，实现100%高成长与高就业。
